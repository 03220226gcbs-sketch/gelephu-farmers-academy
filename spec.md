# Specification

## Summary
**Goal:** Add user progress tracking accounts, a farmer testimonials system, and improve mobile responsiveness across the Gelephu Farmers Academy app.

**Planned changes:**
- Add Internet Identity authentication with a personal dashboard showing enrolled courses, completion status, and earned achievements; progress data persisted per user in the backend
- Add a testimonial submission form (name, location, quote, farming area) with backend storage; admin panel controls for approving/rejecting submissions; approved testimonials displayed on the homepage or a dedicated testimonials page with fallback placeholder content
- Audit and fix mobile responsiveness across all pages: collapse navigation into a mobile menu, fix layout overflow on 320px–768px screens, ensure cards/tables/dashboards stack correctly, set minimum 44×44px touch targets, and make hero sections, forms, and FAQ accordion fully functional on mobile

**User-visible outcome:** Farmers can log in with Internet Identity to track their training progress on a personal dashboard, submit and view community testimonials, and use the full site comfortably on a phone.
